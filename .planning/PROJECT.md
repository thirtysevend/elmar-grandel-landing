# Elmar's Data & AI Training Landing Page

## What This Is

A high-impact, visually stunning single-page landing site for Elmar, a data and AI trainer who runs workshops and courses. The page showcases his expertise through cutting-edge web visuals (3D elements, particle effects, interactive animations) and drives visitors to get in touch. Bilingual (German/English) with a language toggle.

## Core Value

Visitors immediately perceive Elmar as a cutting-edge AI expert through the page's visual wow factor and clear presentation of his training offerings.

- Bilingual audience (German-speaking market + international)

## Constraints

- **Hosting**: Local only for MVP — must work with simple local server or file open
- **Single page**: One landing page, not a multi-page site
- **No backend**: Pure frontend — HTML/CSS/JS with optional lightweight framework
- **Performance**: Heavy visuals must still load and run smoothly
