  <action>
Create all six CSS files implementing the dark futuristic theme, responsive layout, and section styling. Follow mobile-first approach. Use CSS custom properties from tokens.css throughout.

**src/styles/reset.css:**
Use Andy Bell's modern CSS reset (https://piccalil.li/blog/a-more-modern-css-reset/). Key items: box-sizing border-box on all elements, remove default margins, set body min-height 100vh, img max-width 100%, smooth scrolling with prefers-reduced-motion respect.

**src/styles/tokens.css:**
Define ALL design tokens on `:root`. Follow the research palette but adjust to match premium B2B data/AI consultant aesthetic (not startup/gaming). Use:
- Background layers: `--bg-primary: #0a0a0f`, `--bg-secondary: #12121a`, `--bg-surface: #1a1a2e`, `--bg-elevated: #252540`
- Text: `--text-primary: #e0e0e8`, `--text-secondary: #8888a0`, `--text-accent: #ffffff`
- Accent colors: Use a refined palette centered on electric blue/cyan (`--accent-blue: #3b82f6`), warm amber/gold (`--accent-gold: #f59e0b`), and a subtle cyan for glow (`--accent-cyan: #06b6d4`). These feel more B2B-premium than neon pink/purple.
- Glow effects: Subtle box-shadow values for accent-blue and accent-cyan. Keep sophisticated, not flashy.
- Spacing scale: xs through xl using rem
- Typography: `--font-body: 'Inter', system-ui, sans-serif`, `--font-display: 'Space Grotesk', var(--font-body)`. Use `clamp()` for fluid sizing: `--text-sm` through `--text-2xl`.
- Layout: `--nav-height: 64px`, `--content-max-width: 1200px`
- Glassmorphism: `--glass-bg: rgba(10, 10, 15, 0.8)`, `--glass-blur: 12px`, `--glass-border: rgba(255, 255, 255, 0.05)`

**src/styles/layout.css:**
- Set `body` background to `var(--bg-primary)`, color to `var(--text-primary)`, font-family to `var(--font-body)`
- `.section-content` with max-width, auto margins, responsive padding
- Mobile-first breakpoints at 768px and 1024px adjusting padding and layout proportions
- `section[id]` gets `scroll-margin-top: calc(var(--nav-height) + 1rem)` for smooth scroll offset
- `html` gets `scroll-behavior: smooth` wrapped in `@media (prefers-reduced-motion: no-preference)`

**src/styles/nav.css:**
- `.site-header` fixed top, full width, z-index 100, height `var(--nav-height)`, glassmorphism background with `backdrop-filter: blur(var(--glass-blur))`, subtle bottom border
- `.main-nav` flex layout, space-between, center-aligned, max-width constrained, padded
- `.nav-logo` text using display font, accent color, no underline, subtle glow on hover
- `.nav-menu` on desktop: flex row, gap spacing, no bullets. Links styled with text-secondary color, transition to accent-blue on hover, `.active` class gets accent-blue color and subtle underline
- `.nav-toggle` hidden on desktop (display: none above 768px). On mobile: visible, styled as hamburger icon using three-line CSS pattern (::before, span, ::after). No SVG needed.
- Below 768px: `.nav-menu` becomes full-width dropdown below header, hidden by default, shown with `.is-open` class. Background matches header glassmorphism. Links stack vertically with padding.

**src/styles/sections.css:**
- `.section` base: `min-height: 100vh; min-height: 100dvh;` with position relative
- `.section--hero`: flexbox center both axes, text-align center. Background `var(--bg-primary)`. `h1` uses display font, `--text-2xl`, white color, subtle cyan text-shadow glow. `.hero-tagline` uses `--text-lg`, `--text-secondary` color, letter-spacing 0.05em. `.hero-cta` flex row with gap, margin-top. `.btn--primary` with accent-blue background, white text, rounded corners, padding, hover glow. `.btn--secondary` transparent bg, accent-blue border, accent-blue text, hover fills. `.hero-bg` absolute fill, z-index -1 (reserved for Phase 2 animation canvas).
- `.section--expertise`: Background `var(--bg-secondary)`. Top border: 1px gradient line (transparent -> accent-cyan -> transparent) using `::before` pseudo-element with subtle glow. `.section-heading` display font, `--text-xl`, white. `.section-description` `--text-base`, `--text-secondary`, max-width ~600px for readability. `.expertise-placeholder` centered, padded container with dashed border in `--bg-elevated` color, the diamond icon in accent-cyan.
- `.section--contact`: Background `var(--bg-surface)`. Similar heading/description styling. `.contact-placeholder` similar dashed-border placeholder style.
- `.site-footer`: small, subtle, `--text-secondary`, centered, padding, border-top with glass-border color

**src/styles/utilities.css:**
- `.glow-text` helper: text-shadow with accent glow
- `.glow-border` helper: box-shadow with accent glow
- `.fade-in` keyframes animation (opacity 0 -> 1, translateY 20px -> 0) for future use, respects prefers-reduced-motion
- `.visually-hidden` (sr-only) class for accessibility
- Selection color styling (::selection with accent background)

Ensure minimum 4.5:1 contrast ratio for all text/background combinations (WCAG AA). The primary text #e0e0e8 on #0a0a0f gives ~14:1 contrast. Secondary text #8888a0 on #0a0a0f gives ~5.3:1.
