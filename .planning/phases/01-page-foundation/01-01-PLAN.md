Implement the full page structure and dark theme. Follow the patterns from 01-RESEARCH.md exactly.

**index.html** -- Full semantic HTML structure:
- DOCTYPE, html lang="en", head with charset UTF-8, viewport meta, title "Elmar | Data &amp; AI Training"
- Google Fonts preconnect links for Inter (body) and Space Grotesk (display): `<link rel="preconnect" href="https://fonts.googleapis.com">`, `<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>`, then the stylesheet link for `Inter:wght@300;400;600;700` and `Space+Grotesk:wght@400;500;700`
- Script tag: `<script type="module" src="/src/main.js"></script>` in body
- Header with `<nav class="main-nav" aria-label="Main navigation">`: logo text link "Elmar", hamburger button with aria-expanded/aria-controls/aria-label attributes, ul#nav-menu with three li>a items linking to #hero ("Home"), #expertise ("Expertise"), #contact ("Contact")
- Main element with three sections:
  - `<section id="hero" class="section section--hero">` with div.section-content containing h1 "Elmar" and p.hero-tagline "Data & AI Training" and p.hero-subtitle "Expert workshops and courses to transform your team's capabilities"
  - `<section id="expertise" class="section section--expertise">` with div.section-content containing h2 "Expertise" and p "Placeholder for Phase 2 interactive expertise visualization"
  - `<section id="contact" class="section section--contact">` with div.section-content containing h2 "Get in Touch" and p "Placeholder for Phase 3 contact form"
- Footer with copyright text

**src/styles/reset.css** -- Modern CSS reset (Andy Bell style):
- `*, *::before, *::after { box-sizing: border-box; }`
- `body, h1, h2, h3, p { margin: 0; }`
- `html { scroll-behavior: smooth; }` with `@media (prefers-reduced-motion: reduce) { html { scroll-behavior: auto; } }`
- `img, picture, video { max-width: 100%; display: block; }`
- `body { min-height: 100vh; min-height: 100dvh; text-rendering: optimizeSpeed; line-height: 1.6; }`

**src/styles/tokens.css** -- CSS custom properties on :root, exactly as specified in RESEARCH Pattern 1:
- Background colors: --bg-primary (#0a0a0f), --bg-secondary (#12121a), --bg-surface (#1a1a2e), --bg-elevated (#252540)
- Text colors: --text-primary (#e0e0e8), --text-secondary (#8888a0), --text-accent (#ffffff)
- Accent/neon: --accent-cyan (#00f0ff), --accent-purple (#8b5cf6), --accent-pink (#ff00ff)
- Glow effects: --glow-cyan, --glow-purple (box-shadow values from research)
- Spacing scale: --space-xs through --space-xl
- Typography: --font-body (Inter + system fallbacks), --font-display (Space Grotesk + fallback), --text-sm through --text-2xl using clamp()
- Layout: --nav-height (64px), --content-max-width (1200px)

**src/styles/layout.css** -- Responsive layout:
- `section[id] { scroll-margin-top: calc(var(--nav-height) + var(--space-sm)); }`
- Mobile-first .section-content with padding, max-width, margin auto
- `@media (min-width: 768px)` breakpoint with tablet padding
- `@media (min-width: 1024px)` breakpoint with desktop padding

**src/styles/nav.css** -- Fixed glassmorphism navigation:
- .site-header: fixed, top 0, full width, z-index 100, height var(--nav-height), background rgba(10,10,15,0.8), backdrop-filter blur(12px), border-bottom 1px solid rgba(255,255,255,0.05)
- .main-nav: flex, align-items center, justify-content space-between, padding inline
- .nav-logo: font-family display, font-size text-lg, color text-accent, text-shadow glow-cyan, text-decoration none
- .nav-menu: flex, list-style none, gap spacing. Links styled with text-secondary color, transition on color, hover/active states using accent-cyan
- .nav-toggle: hidden on desktop (display: none above 768px), styled hamburger spans below 768px
- Mobile: @media (max-width: 767px) -- .nav-menu hidden by default, shown when .is-open class added, vertical flex column, full-width overlay or dropdown with bg-secondary background

**src/styles/sections.css** -- Section-specific styles (use ONLY var() references for colors, never raw hex):
- .section--hero: min-height 100vh / 100dvh, flex center, text-align center, relative, overflow hidden, background var(--bg-primary)
- .section--hero h1: font-display, text-2xl, color text-accent, text-shadow glow-cyan, letter-spacing 0.05em
- .hero-tagline: font-display, text-xl, color accent-cyan
- .hero-subtitle: text-base, color text-secondary, max-width 600px, margin auto
- .section--expertise: background var(--bg-secondary), min-height 80vh, position relative
- .section--expertise::before: neon accent border at top (gradient line from research Pattern)
- .section--contact: background var(--bg-primary), min-height 60vh
- Section headings: font-display, text-xl, color text-accent, margin-bottom

**src/styles/utilities.css** -- Utility classes:
- .glow-text: text-shadow glow-cyan
- .glow-border: box-shadow glow-cyan
- .glass: backdrop-filter blur, semi-transparent background, border
- Subtle fade-in animation keyframes (respect prefers-reduced-motion)

**src/main.js** -- Import all CSS files:
```javascript

import './styles/sections.css';
```
